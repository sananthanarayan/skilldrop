## How to read this report

This report shows **patterns**, not verdicts. A user with a low effectiveness rate may be:

- Exploring (high volume of throwaway prompts while learning a tool)
- Solving in one shot (one prompt is enough — no follow-up needed)
- Doing AI theater (invoking to be seen invoking)

The data alone cannot distinguish these. Use the report to surface conversations, not to grade individuals.
